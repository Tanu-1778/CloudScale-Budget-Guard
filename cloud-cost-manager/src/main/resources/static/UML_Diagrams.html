2. Sequence Diagram (User Login Flow)

Illustrates the step-by-step communication during the authentication process.

                sequenceDiagram
                    participant User
                    participant React_Frontend
                    participant SpringBoot_Controller
                    participant AppUser_Service
                    participant H2_Database

                    User->>React_Frontend: Enters credentials & clicks Login
                    React_Frontend->>SpringBoot_Controller: POST /api/auth/login (JSON)
                    SpringBoot_Controller->>AppUser_Service: login(username, password)
                    AppUser_Service->>H2_Database: SELECT * FROM app_users WHERE username = ?
                    H2_Database-->>AppUser_Service: User Record Found
                    AppUser_Service-->>SpringBoot_Controller: UserDTO (Success)
                    SpringBoot_Controller-->>React_Frontend: 200 OK + Profile Data
                    React_Frontend-->>User: Redirect to Dashboard
